# LZT Control

LZT Control — локальная панель для работы с объявлениями и покупками LZT Market. Она объединяет поднятие объявлений, проверки аккаунтов, публикацию после гарантии, контроль API и полезные инструменты в одном интерфейсе.

## Возможности

- автоматическое и ручное поднятие объявлений;
- отдельные задачи, лимиты и расписания для нескольких LZT-аккаунтов;
- плановые проверки покупок во время гарантии;
- проверка валидности и Steam-ограничений;
- ручная и автоматическая публикация аккаунтов после гарантии;
- история операций, журнал запросов и понятная обработка ошибок;
- монитор лимитов LZT API;
- аудит покупок и перепродаж;
- проверка Steam-аккаунтов на KT;
- анализ цен продажи и предложений скупщиков;
- массовое создание арбитражей;
- светлая, тёмная и LZT-тема интерфейса.

## Что понадобится

- Windows 10 или Windows 11;
- [Python 3.11 или новее](https://www.python.org/downloads/windows/);
- подключение к интернету;
- API-токен LZT Market.

При установке Python рекомендуется включить пункт `Add Python to PATH`.

## Запуск

1. На странице проекта в GitHub нажмите `Code`, затем `Download ZIP`.
2. Полностью распакуйте скачанный архив в отдельную папку.
3. Дважды нажмите `start.bat`.
4. При первом запуске дождитесь автоматической установки компонентов.
5. Панель откроется в браузере по адресу `http://127.0.0.1:8787`.

Первый запуск может занять несколько минут. В дальнейшем приложение запускается тем же файлом `start.bat` значительно быстрее.

Не закрывайте окно LZT Control во время работы — в нём запущен локальный сервер приложения.

## Первоначальная настройка

После запуска откройте менеджер аккаунтов в правой части верхней панели и добавьте API-токен LZT. Для каждого аккаунта можно задать имя, цвет, собственный суточный лимит и прокси.

Остальные параметры находятся в настройках соответствующих разделов:

- в разделе «Поднятие» настраиваются задачи и расписание;
- в разделе «Проверка» задаются ссылка на покупки, этапы проверок и пролив;
- Steam Web API и Telegram подключаются только при необходимости;
- утилиты открываются через вкладку «Утилиты».

## Завершение работы

Закройте окно консоли LZT Control или нажмите в нём `Ctrl+C`. После завершения процесса локальная страница перестанет открываться.

## Если приложение не запускается

- Убедитесь, что установлена поддерживаемая версия Python.
- Проверьте подключение к интернету: при первом запуске загружаются необходимые компоненты.
- Если браузер не открылся автоматически, перейдите на `http://127.0.0.1:8787` вручную.
- Если указано, что порт `8787` уже занят, закройте ранее запущенное окно LZT Control и повторите запуск.
- Если локальное окружение было повреждено, удалите папку `.venv` и снова запустите `start.bat`.

## Безопасность и приватность

LZT Control работает локально и доступен только по адресу `127.0.0.1`. Пользовательские данные не отправляются разработчику. Сетевые запросы выполняются только к сервисам, необходимым выбранным функциям: LZT Market, Steam и Telegram.

API-токены, Steam-ключи, данные Telegram и реквизиты прокси защищаются средствами Windows DPAPI. Рабочая база, настройки, журналы и результаты утилит хранятся только на компьютере пользователя.

Не передавайте другим людям свою рабочую папку приложения после добавления аккаунтов. Для установки на другом компьютере используйте исходный чистый архив.
